enum SPI_IF_TYPE

Interface type for the spi.

Enumerator
SPI_IF_TYPE_REG 

Access registers with spi

SPI_IF_TYPE_HS 

Preloads reg address to 0 for high speed tests

SPI_IF_TYPE_ECHO 

Echos SPI bytes

SPI_IF_TYPE_CONST 

Always output user reg 0

Precondition

Returns
EOK if init occurred
ENOACTION if no init was triggered
Note
Only executes actions if the i2c.mode.init is set.
